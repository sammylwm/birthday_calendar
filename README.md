# Birthday Calendar

**Birthday Calendar** is a Flutter application that helps you keep track of birthdays and never miss an important celebration.

The app allows you to store birthdays, organize them in one place, and synchronize them with Google Calendar so you can receive reminders using your existing calendar notifications.

## Features

* 🎂 Store and manage birthdays.
* 📅 Sync birthdays with Google Calendar.
* 🔔 Receive birthday reminders through Google Calendar notifications.
* ✏️ Edit or remove birthdays at any time.
* 📱 Clean, simple, and intuitive user interface.
* ⚡ Built with Flutter for a smooth cross-platform experience.

Whether you're keeping track of family, friends, or colleagues, Birthday Calendar makes it easy to stay organized and remember every special day.
