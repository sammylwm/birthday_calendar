Birthday Calendar

Birthday Calendar is a mobile application that helps users organize and manage birthday events.

About the application

Birthday Calendar allows users to view upcoming birthdays, create birthday events in Google Calendar, and keep important dates organized.

Google Calendar integration

The application uses Google Calendar API to synchronize birthday events. With user permission, Birthday Calendar can access calendar events and create birthday events in the user's Google Calendar.

The application only accesses Google Calendar data required for birthday management features.

Features

  • View upcoming birthdays
  • Create birthday events in Google Calendar
  • Manage birthday reminders
